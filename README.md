C++ control system for quadrotor attitude, altitude, and lateral position control using cascaded PID controllers and body rate commands.

---

## Key Concepts

- Cascaded PID control architecture
- Body rate control
- Altitude controller
- Lateral position controller
- Yaw control
- Motor mixing algorithms
